<Info>
  To enable an agent to send messages through Intercom, you need to include the
  admin ID in the agent's system message or request. The agent will send
  messages on behalf of the admin whose ID is used.
</Info>

Finding your Intercom Admin ID is a simple process:

<Steps>
  <Step title="Check URL in Your Inbox">
    Navigate to "Your Inbox" in Intercom. Look at the URL in your browser - your
    Admin ID is the number that appears after "admin/" in the URL path.
    <Frame>
      <img
        src="/how-to/finding-ids/intercom1.png"
        alt="Admin ID in Intercom URL"
      />
    </Frame>
  </Step>
</Steps>

For example, an admin ID might look like: `1234567`
