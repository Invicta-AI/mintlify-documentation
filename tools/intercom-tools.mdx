This section provides an overview of the Intercom tools available for AI Agents to interact with Intercom. These tools allow agents to perform operations such as sending messages within the Intercom ecosystem.

<Note>
  Integration required. [Learn how to set up.](/how-to/integration-setup)
</Note>

## List of Intercom Tools

| Tool Name             | Description                                              |
| --------------------- | -------------------------------------------------------- |
| Intercom Send Message | Allows AI agents to send messages using the Intercom API |

## Intercom Send Message

### Description

This tool enables AI agents to send messages through Intercom's messaging system, facilitating automated customer support interactions.

### System Tool ID and Tool Description for Agents

- **Tool ID:** `intercom_send_message`
- **Description:** Send message using Intercom.

### Arguments

| Name            | Required | Type   | Description                  |
| --------------- | -------- | ------ | ---------------------------- |
| admin_id        | Yes      | String | Intercom User Id for replier |
| conversation_id | Yes      | String | Conversation ID              |
| body            | Yes      | String | Content of the message       |
